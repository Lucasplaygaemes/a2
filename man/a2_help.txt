# a2 - A Modal Text Editor

Welcome to `a2`, a lightweight, fast, and configurable text editor inspired by Vim.

## Quick Navigation
- For help with *commands* (the ':' mode), see the commands manual: [*Commands*](commands_help.txt)
- For help with *keyboard shortcuts*, see the shortcuts manual: [*Shortcuts*](shortcuts_help.txt)

## Core Features

- *Configuration:* All settings, macros, and keyboard shortcuts are stored centrally in `~/.a2/`, making your environment easy to backup and portable.
- *Spell Checker:* Real-time spell checking with suggestions (Tab completion) and hover-info for misspelled words. Activated conditionally for .txt files or when no Language Server Protocol (LSP) is active.
- *Modal Editing:* Efficiently edit text with Normal, Insert, and Visual modes.
- *Workspaces & Splits:* Organize your work across multiple workspaces, each with its own layout of split windows.
- *Fuzzy Finder:* Quickly find and open any file in your project with a fuzzy search (*Alt+F*).
- *Command Palette:* A searchable command palette (*Alt+T*) for easy access to all editor commands.
- *File Explorer:* A built-in file explorer (*:explorer*) to navigate and manage files.
- *Atomic Save:* Writes files safely to prevent corruption in the event of power loss or a crash, while maintaining the automatic `.sv` backup.
- *LSP Support:* Language Server Protocol integration for real-time diagnostics, code highlighting, and completion.
- *Interactive Popups:* Diagnostic and spell checking hovers can become movable floating windows (use *Ctrl+Shift+M* or click them). Press Enter to pin them to the screen.
- *Macros:* Record and play back sequences of commands (*q* and *@*) to automate repetitive tasks.
- *Advanced Search & Replace:* A powerful *:s* command that supports regular expressions for text substitutions.
- *Unified Search:* Search with */* or *Ctrl+F* using the unified command bar with full history and arrow navigation.
- *Theming:* Customize the editor's appearance with simple `.theme` files.
- *Assembly View:* Compile and view assembly code synchronized with your C source (*Alt+A*).
- *Snippets:* Expand code snippets from the completion menu (*Alt+S* during completion).
- *Project Management:* Save and load project sessions, including open files, window layouts, and cursor positions.
- *Custom Tasks:* Define commands and assign them customized keybindings to quickly automate your workflow.
